# Pre-Submission Checklist

## 投稿前检查清单

本清单供 word-submit 模块在投稿准备阶段使用。分为自动检查（由工具完成）和手动检查（提醒用户确认）两部分。

---

## 自动检查项（word-submit 执行）

### 文档清洁度

| # | 检查项 | 检查方法 | 通过标准 |
|---|--------|---------|---------|
| A1 | 无未接受的修订 | 扫描 XML 中的 `<w:ins>` / `<w:del>` | 零个修订标记 |
| A2 | 无批注 | `get_all_comments` 返回空 | 零条批注 |
| A3 | 无个人元数据 | 读取 `docProps/core.xml` | creator/lastModifiedBy 为空 |
| A4 | 文件大小合理 | 检查文件大小 | < 期刊限制（通常 10-50MB） |

### 交叉引用完整性

| # | 检查项 | 检查方法 | 通过标准 |
|---|--------|---------|---------|
| B1 | 图编号连续 | word-checker R2 | 从1开始连续递增 |
| B2 | 表编号连续 | word-checker R2 | 从1开始连续递增 |
| B3 | 无悬空引用 | word-checker R1 | 所有引用指向存在的资产 |
| B4 | 无孤立资产 | word-checker R6 | 所有图表在正文中被引用 |
| B5 | 引用格式一致 | word-checker R5 | 全文统一使用中文或英文格式 |

### 格式合规性

| # | 检查项 | 检查方法 | 通过标准 |
|---|--------|---------|---------|
| C1 | 页面设置 | word-checker Mode B | 符合格式要求 |
| C2 | 字体字号 | word-checker Mode B | 符合格式要求 |
| C3 | 行距段距 | word-checker Mode B | 符合格式要求 |
| C4 | 页码 | word-checker Mode B | 符合格式要求 |

### 字数统计

| # | 检查项 | 检查方法 | 通过标准 |
|---|--------|---------|---------|
| D1 | 总字数 | 统计全文字数 | 在期刊限制内 |
| D2 | 摘要字数 | 统计摘要段落字数 | 在期刊限制内（通常 150-300 词） |
| D3 | 关键词数量 | 统计关键词 | 在期刊要求范围内（通常 3-8 个） |

---

## 手动检查项（提醒用户确认）

### 内容完整性

| # | 检查项 | 说明 |
|---|--------|------|
| E1 | 标题准确 | 标题是否反映论文核心内容 |
| E2 | 作者信息正确 | 所有作者姓名、单位、邮箱是否正确 |
| E3 | 通讯作者标注 | 通讯作者是否明确标注 |
| E4 | 摘要完整 | 中英文摘要是否都包含（如适用） |
| E5 | 关键词合适 | 关键词是否覆盖论文主题 |
| E6 | 致谢完整 | 基金资助、人员致谢是否完整 |
| E7 | 利益冲突声明 | 是否包含利益冲突/利益相关声明 |
| E8 | 数据可用性声明 | 数据和代码的可获取性声明 |
| E9 | 伦理声明 | 涉及人类/动物研究时的伦理批准信息 |

### 图表质量

| # | 检查项 | 说明 |
|---|--------|------|
| F1 | 图片分辨率 | 所有图片 ≥ 300 dpi（需在专业软件中确认） |
| F2 | 图片格式 | 是否符合期刊要求（TIFF/EPS/PDF/PNG） |
| F3 | 图表可读性 | 字体大小在缩放后是否仍可读 |
| F4 | 色彩方案 | 是否考虑色觉障碍读者（避免仅用红绿区分） |

### 投稿材料

| # | 检查项 | 说明 |
|---|--------|------|
| G1 | Cover letter | 是否已准备投稿信 |
| G2 | 推荐审稿人 | 是否已准备推荐/回避审稿人名单 |
| G3 | 补充材料 | 补充材料是否已单独准备 |
| G4 | 图表单独文件 | 部分期刊要求图表单独上传 |
| G5 | 数据文件 | 附带的数据集/代码是否已准备 |

---

## 报告格式

word-submit 执行检查后，输出格式：

```
## 投稿前检查报告: {filename}

### 自动检查 ({passed}/{total})
- ✅ A1: 无修订痕迹
- ✅ A2: 无批注
- ✅ A3: 无个人元数据
- ✅ B1-B5: 交叉引用完整
- ✅ C1-C4: 格式合规
- ✅ D1: 总字数 8,234 (限制: 10,000)
- ✅ D2: 摘要 248 词 (限制: 250)
- ❌ D3: 关键词 9 个 (限制: 3-8) → 需减少1个

### 请手动确认
- [ ] E1: 标题准确
- [ ] E2: 作者信息正确
- [ ] E3: 通讯作者已标注
...
```
