# Academic Formatting Reference

## 通用学术论文格式规范

本文件提供学术论文的通用格式规范知识，供 word-formatter 在解析和执行格式要求时参考。这些是**通用知识**，不是默认值——实际格式以用户提供的要求文档为准。

## 页面设置常见规格

| 项目 | 中文期刊常见 | 英文期刊常见 |
|------|------------|------------|
| 纸张 | A4 (210×297mm) | A4 或 Letter (8.5×11in) |
| 上下边距 | 2.54cm | 2.54cm (1in) |
| 左右边距 | 3.17cm 或 2.54cm | 2.54cm (1in) |
| 页码位置 | 底部居中 | 底部居中或右下 |

## 字体常见搭配

### 中文论文
| 元素 | 中文字体 | 英文字体 | 常见字号 |
|------|---------|---------|---------|
| 论文标题 | 黑体 | Arial | 二号-三号 (22-16pt) |
| 一级标题 | 黑体 | Arial | 三号-四号 (16-14pt) |
| 二级标题 | 黑体 | Arial | 四号-小四 (14-12pt) |
| 三级标题 | 楷体 或 黑体 | Arial | 小四 (12pt) |
| 正文 | 宋体 | Times New Roman | 小四-五号 (12-10.5pt) |
| 图表题注 | 宋体 | Times New Roman | 五号 (10.5pt) |
| 参考文献 | 宋体 | Times New Roman | 五号 (10.5pt) |
| 页眉/页脚 | 宋体 | Times New Roman | 五号 (10.5pt) |

### 英文论文
| 元素 | 字体 | 常见字号 |
|------|------|---------|
| Title | Times New Roman / Arial | 14-16pt |
| Heading 1 | Times New Roman / Arial | 12-14pt, Bold |
| Heading 2 | Times New Roman / Arial | 12pt, Bold or Italic |
| Body | Times New Roman | 12pt |
| Caption | Times New Roman / Arial | 10pt |
| References | Times New Roman | 10-12pt |

## 行距规范

| 场景 | 常见值 |
|------|-------|
| 投稿版正文 | 双倍行距 (2.0) — 方便审稿 |
| 终稿正文 | 1.5 倍行距 |
| 参考文献 | 单倍行距 (1.0) |
| 图表题注 | 单倍行距 (1.0) |
| 摘要 | 与正文相同 或 单倍行距 |

## 标题编号格式

### 中文常见格式
```
一、XXX          （一级，中文数字）
（一）XXX        （二级，带括号中文数字）
1. XXX           （三级，阿拉伯数字）
（1）XXX         （四级）
```

或阿拉伯数字层级格式：
```
1 XXX            （一级）
1.1 XXX          （二级）
1.1.1 XXX        （三级）
```

### 英文常见格式
```
1. XXX           (Level 1)
1.1 XXX          (Level 2)
1.1.1 XXX        (Level 3)
```

## 图表规范

### 三线表（Three-Line Table）
学术论文最常用的表格样式，只有三条横线：
- **顶线**（top border）：表格顶部，较粗（1-1.5pt）
- **栏目线**（header border）：表头与数据之间，较细（0.5-0.75pt）
- **底线**（bottom border）：表格底部，较粗（1-1.5pt）
- 无竖线，无其他横线
- 表题在表格上方
- 表注在表格下方

### 图片规范
- 分辨率：≥300 dpi（印刷质量）
- 图题在图片下方
- 编号格式：与正文引用一致（"图1" 或 "Figure 1"）

## 参考文献格式

### 中文论文常见格式
- **GB/T 7714-2015 顺序编码制**：`[1] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.`
- **GB/T 7714-2015 著者-出版年制**：`(作者, 年份)`

### 英文论文常见格式
- **APA 7th**：`Author, A. A. (Year). Title. Journal, Volume(Issue), Pages.`
- **Harvard**：`Author (Year) Title. Journal, Volume, Pages.`
- **Vancouver (Numbered)**：`[1] Author. Title. Journal. Year;Volume(Issue):Pages.`

## 页眉页脚

| 类型 | 常见内容 |
|------|---------|
| 页眉 | 论文简短标题 或 期刊名称 |
| 页脚 | 页码（阿拉伯数字，从正文开始） |
| 首页 | 通常无页眉，可能有特殊页脚 |

## 首行缩进

| 语言 | 规范 |
|------|------|
| 中文 | 首行缩进 2 字符（约 24pt 或 0.85cm） |
| 英文 | 首行缩进 0.5in (1.27cm) 或无缩进+段间距 |

## 注意事项

- 以上均为**通用参考知识**，不作为默认格式使用
- 实际格式以用户提供的格式要求文档为准
- 当格式要求与通用规范冲突时，以用户要求为准
- 当格式要求未涉及某项时，保持文档原样，不自动套用通用规范
